# 一、 pass君 🎀

pass君是一款可以跳过开屏广告的安卓手机软件工具。
目前只支持部分文本匹配，变速功能暂时未完成。
变速打算用hook实现的，用模拟器做测试的时候发现许多广告软件都有模拟器签名校验，
打算用真机测试的又发现使用面具的xposed模块需要需要刷root权限，
刷机可能变砖🧱，项目就暂时搁置了.
进修一下逆向和变速器原理后以后可能会更新。
